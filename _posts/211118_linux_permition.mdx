## 概要
- chmod コマンドに関して再復習

## 使用環境

- macOS, Terminal
- shell: zsh

## コマンド
```
  chmod [モード] [ファイル]
  # 例
  chmod +x example.txt
```
## モード
- 「１：誰に対して」「２：どうするか」「３：どの動作に」を記述
- 「シンボルモード」「数値モード」で記述
- まずは「シンボルモード」を前提に説明

### １：誰に対して
- 「シンボルモード」
- u: オーナー
- g: グループ
- o: その他ユーザー
- a: ugo 全て


```
    # 例: example.txt の実行権限をオーナーに付与
    chmod u+x example.txt
```

### ２：どうするか
- 権限をどうするか
- +: 付与
- -: 剥奪
- =: 指定権限と等しくする

```
    # 例: example.txt の書き込みをグループに対して無効化
    chmod g-w example.txt
```

### ３：どの動作に
- 動作の対象を指定
- r: 読込
- w: 書込
- x: 実行

```
  # 例: example.txt の実行権限を全てのユーザーに付与
  chmod a+x example.txt
```
