# JamStack なブログのススメ

「概要はいいから、作り方はよ！」という方は、「環境構築」からどうぞ。

## Jamstack とは？

Webサイトを構築・運用に用いる主要な使用技術 JavaScript, API, Markdown のこと。
記事は Markdown でかけることから、特にエンジニアには特におすすめ。

## なぜ Jamstack を推すのか
- やすい
- ビルドがはやい(特に Next.js)

### やすい
かかった費用：ドメイン代 のみ。

- ホスティングサービスはどれを選べばいいか？
  - Vercel：手軽。無料版は商用不可(アフィリエイトのリンク NG)
  - Cloudflare Pages：一手間かかるが、ビルド時間の制限の上限などが一番高め。
  - Netlify
  - GitHub Pages

### ビルドがはやい(特に Next.js)

dev 環境が立ち上がるのに 7秒くらい
JamStack ブログでの対抗馬である、Gatsuby.js と比べると特に実感できる。 

多機能な「Gatsuby」, シンプル・サクサクな「Next.js」かなというのが自分の意見です。

- デメリット

Next.js のJamstack関連の日本語情報が少ない。
